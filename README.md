# coolify-deploy

Trigger a Coolify deployment and VALIDATE it — poll the build to a terminal state, then check app health, and report PASS/FAIL with logs on failure

## Install (Claude Code)

```
/plugin marketplace add jachana/claude-marketplace
/plugin install coolify-deploy@jachana
```

Part of the [jachana marketplace](https://github.com/jachana/claude-marketplace).
